## מה זה

שאלון של 8 שאלות שמודד את המוכנות הניהולית של ארגון לאירוע סייבר. בסוף השאלון המנהל מקבל ציון, ממצאים, ו-PDF להורדה. הנתונים נשלחים אוטומטית ל-Make webhook ומשם ל-Google Sheets.

## טכנולוגיה

- HTML סטטי — ללא שרת
- jsPDF — יצירת PDF בדפדפן
- Make webhook — תיעוד כל מילוי ב-Google Sheets

## קהל יעד

- גרסה אנגלית — ארה"ב ואירופה
